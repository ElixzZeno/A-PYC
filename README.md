```bash
apt install cmake
```

apt install clang
```

apt install make
```
```bash
apt install git
```

## Usage

Clone the ElixzZeno repository from GitHub:

```bash
git clone https://github.com/ElixzZeno/A-PYC.git
```

Navigate to the AsmitX directory:

```bash
cd A-PYC
```

Run the script:

```bash
python run_enc.py
```
